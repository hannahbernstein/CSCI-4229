# HW2- Lorenz Attractor


Demonstrates how to draw a 3D Lorenz Attractor

Key bindings
  *m/M        Cycle through different sets of objects\
  *a          Toggle axes\
  *arrows     Change view angle\
  *0          Reset view angle\
  *ESC        Exit\\
